Overview

This IOI Algorithms Preparation Guide is designed to help students prepare for the International Olympiad in Informatics (IOI) and other competitive programming contests. The guide covers essential algorithms and data structures that are frequently tested in such competitions.

The guide is structured to provide a comprehensive understanding of various algorithmic concepts, starting from basic techniques and progressing to more advanced topics. Each section includes theoretical explanations, example problems, and practice exercises to reinforce learning.

Topics Covered
  • Graph Algorithms: BFS, DFS, Shortest Paths, Minimum Spanning Trees
  • Dynamic Programming: Classical Problems, State Optimization, Bitmask DP
  • Data Structures: Segment Trees, Fenwick Trees, Disjoint Set Union
  • Number Theory: Prime Factorization, Modular Arithmetic, GCD/LCM
  • String Algorithms: KMP, Z-Algorithm, Suffix Arrays
  • Geometry: Convex Hull, Line Intersection, Point in Polygon
Prerequisites

Before starting this guide, you should have:

  • Basic programming skills in C++ or Python
  • Understanding of time and space complexity analysis
  • Familiarity with basic data structures (arrays, linked lists, stacks, queues)
  • Experience with recursive problem solving
How to Use This Guide

We recommend following these steps for optimal learning:

  1. Read the theoretical explanation for each topic carefully
  2. Study the example problems and their solutions
  3. Attempt the practice exercises on your own
  4. Review your solutions and compare with optimal approaches
  5. Participate in online judge contests to test your skills
Competition Format

The IOI typically consists of two competition days, each with 3 problems to solve in 5 hours. Problems are graded based on the number of test cases passed, with partial scoring available for most problems.

This guide will help you develop the problem-solving skills and algorithmic knowledge needed to excel in such competitions. Regular practice and consistent effort are key to success!

Problem: Find the length of the longest strictly increasing subsequence in an array of integers.

Input: Array of N integers

Expected Time Complexity: O(N log N)

Problem: Build a segment tree to answer range sum queries and point updates efficiently.

Input: Array of N integers and Q queries

Expected Time Complexity: O(N + Q log N)

Problem: Implement the KMP algorithm to find all occurrences of a pattern in a text string.

Input: Text string and pattern string

Expected Time Complexity: O(N + M)

Problem: Find the shortest path from a source vertex to all other vertices in a weighted graph with non-negative edges.

Input: Graph with V vertices and E edges, source vertex

Expected Time Complexity: O((V + E) log V)
